mysql的安装和配置

软件下载

  1. Mysql5.7软件下载
  2. 下载后会得到zip 安装文件
  3. 解压的路径最好不要有中文和空格
  4. 我解压到 D:\mysql-5.7.19-winx64 目录下 【根据自己的情况来指定目录,尽量选择空间大的盘】

    update user set authentication_string=password(‘sdy’) where user=’root’ and Host=’localhost’;
