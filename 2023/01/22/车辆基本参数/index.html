基本介绍选购传统燃油车需要关注和了解的参数

# 一、发动机

# 1、排量

排量，液压传动专用术语，是指每行程或每循环吸入或排出的流体体积。

汽车排量是衡量一辆汽车动力和油耗的重要参数。

排量划分

中国轿车级别的划分是根据排量的大小来决定的。

微型轿车的排量小于等于 1.0L；

普通级轿车的排量在 1.0—1.6L 范围内；

中级轿车的排量在 1.6—2.5L 范围内；

中高级轿车的排量在 2.5—4.0L 范围内；

高级轿车的排量则大于 4.0L。

目前 1.6 和 2.0L 的车型时市场最大

# 2、进气方式

发动机的动力来自于汽油的燃烧，燃烧是需要空气的，因此发动机不仅需要汽油还需要空气，根据不同的空气供给方式，将发动机分为自然吸气和涡轮增压。

# 2.1、自然吸气

所谓的自然吸气就是外界空气直接与喷油针喷出的汽油混合而形成汽油雾，最后进入燃烧室燃烧。

优点

  • 成本偏低
  • 油耗稳定
  • 技术更成熟，故障率低
  • 对油品要求低
  • 提速线性
  • 维护保养成本低

缺点

  • 同等排量下动力不如涡轮增压
  • 废气排量大

# 2.2、涡轮增压

通过空气压缩器压缩废气来增加内燃机中的空气流量，快速与汽油混合，同排量下带来更大的发动机功率和扭矩。

优点

  • 动力更强劲
  • 尾气排放量少

缺点

  • 维护保养成本高
  • 低转速下存在涡轮延迟现象
  • 强劲动力的同时容易产生爆震
  • 同排量下油耗高于自然吸气
  • 对汽油的种类和辛烷值要求较高
  • 对机油需求量高

# 2.3、汽缸

1~2.5 升一般为 4 缸发动机，3 升左右的发动机一般为 6 缸，4 升左右为 8 缸，5.5 升以上用 12 缸发动机

一般来说，在同等缸径下，缸数越多，排量越大，功率越高；在同等排量下，缸数越多，缸径越小，转速可以提高，从而获得较大的提升功率。

建议选择 4 缸，技术比较成熟，市场认可度高

# 2.4、排放标准

一般来说，在同等缸径下，缸数越多，排量越大，功率越高；在同等排量下，缸数越多，缸径越小，转速可以提高，从而获得较大的提升功率。
